# Google Web Store and Firefox Addons link soon!

## How to manually install for Chrome
- Download the extension (Click on the blue "Code" button and then "Download ZIP")
- Open Chrome and type "chrome://extensions/"
- Enable "Developer mode"
- Click on "Load unpacked"
- Select the ZIP file
- Done!

## How to manually install for Firefox
- Download the extension (Click on the blue "Code" button and then "Download ZIP")
- Open Firefox and type "about:debugging"
- Click on "This Firefox"
- Click on "Load Temporary Add-on..."
- Select the ZIP file
- Done! You must do this every time you reopen Firefox
